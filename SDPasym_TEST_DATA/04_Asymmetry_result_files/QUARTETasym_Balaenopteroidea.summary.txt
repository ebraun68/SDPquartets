TAXON INVOLVEMENT IN SIGNIFICANT ASYMMETRIC QUARTETS (BH CORRECTED)
==============================================================================

Correction method: BH
Alpha level:       0.05

Direction interpretation:
  Positive bias (+): taxon in quartets where ac|bd > ad|bc
    (left-side taxa cluster with group_c more than group_d)
  Negative bias (-): taxon in quartets where ad|bc > ac|bd
    (left-side taxa cluster with group_d more than group_c)

Column descriptions:
  ac|bd / ad|bc  Quartets where each minority topology is favored
  Bias           (ac|bd - ad|bc) / (ac|bd + ad|bc); range [-1, +1]
  R1             Mean(n_favored_minority / n_disfavored_minority);
                 direction set by taxon-level lean.
                 inf(N): all N quartets had disfavored minority = 0
                 X.XX(N): mean of included quartets; N excluded
  R2             Mean(n_favored_minority / n_ab_cd);
                 X.XX(N): N quartets excluded where n_ab_cd = 0
  >maj           Quartets where n_favored_minority > n_ab_cd

The binomial test P-value tests for a directional bias across
all significant quartets. Fewer than 6 significant quartets
gives insufficient power to assess directionality.
